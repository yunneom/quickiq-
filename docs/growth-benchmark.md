### 바이럴 메커니즘 (Woobox·Fyrebox·BuzzFeed)
- 3요소 필수: ① **갖고 싶은 정체성** ② **스크린샷할 결과 이미지** ③ **닫기보다 쉬운 공유 경로**
- **사전작성 공유문구** (유형명 박힘, "I took a quiz" 금지)
- **유형별 커스텀 이미지** (텍스트 결과는 공유 안 됨)
- **친구 태그 프롬프트** ("반대 유형일 친구에게 보내기")
- **비교 후크** ("상위 12%만 이 결과")
- **결과 페이지 선택적 이메일 수집** (사전 게이트 X — 완주율 반토막)
- 완주율이 공유의 관문 — 결과를 못 보면 공유도 없음

## 2. 적용 완료 (이번 커밋) — 결제 무관, 100% 코드

### A. 프로그래매틱 SEO 유형 페이지 ⭐ 최대 레버
16personalities·thicket.sh의 #1 성장 엔진을 그대로 적용.
- **38개 유형 × 2언어 = 76개 정적(SSG) 페이지** 신규 생성
  - `/[locale]/mbti/types/[type]` (16) · `/enneagram/types/[type]` (9)
  - `/love-lang/types/[type]` (5) · `/attachment/types/[type]` (4)
  - `/teto-egen/types/[type]` (4)
- 각 페이지: 유형 설명 + 강점/약점 + 궁합 + **Article JSON-LD** + 테스트 CTA
- **교차링크**: 같은 테스트의 다른 유형(롱테일 키워드) + 다른 테스트(탐색 루프)
- **sitemap.xml에 76개 유형 URL 자동 등록** (hreflang alternates 포함)

### B. 유형별 공유 카드 (커스텀 이미지)
"유형별 커스텀 이미지" 레버 — 공유 시 유형명이 박힌 1200×630 OG 자동 생성.
- 각 유형 페이지 + 결과 페이지 공유 시 해당 유형 카드 노출
- `lib/og/test-og.tsx`의 `renderProfileOg`로 생성

### C. 결과 페이지 바이럴 업그레이드 (4개 테스트)
- **사전작성 공유문구**: `내 결과는 "INFP · 중재자" 나왔어! 너도 해볼래?`
- **친구 태그 프롬프트**: "이 유형일 것 같은 친구에게 보내보세요"
- **네이티브 공유 + 링크 복사** (navigator.share → 클립보드 폴백)
- **유형 상세 deep link**: 결과 → 정식 유형 페이지(공유·SEO 자산)
- **교차 테스트 추천**: 결과 하단에 다른 테스트 3개 카드 (탐색 루프)

### D. 허브 OG / 테스트별 OG (앞선 커밋)
- Hub + 6개 테스트 각각 고유 OG 1200×630
- 카톡·X·페이스북 공유 시 자동 노출

## 3. 대기 (의존성 있음 — 해제되면 즉시 적용)

| 레버 | 의존성 | 메모 |
|---|---|---|
| **결과 페이지 광고(AdSense)** | 광고 승인 | 푸망식 "결과=광고지면". 무료 사용자 ARPU. 트래픽 쌓인 뒤. |
| **유료 상세 리포트(성격)** | 토스 결제 | IQ처럼 성격 테스트도 PDF 유료화 가능. |
| **환불 보장 카피** | 결제 | 16personalities식 "30일 환불 보장"으로 전환율↑. |
| **결과 저장 + 이메일 수집** | Supabase | 결과 페이지 선택적 이메일(가치교환). 리텐션·재방문. |
| **GSC sitemap 제출** | 운영자 OAuth | 색인 가속. 배포 후 Search Console에 sitemap.xml 등록 필요. |
| **친구 궁합 매칭** | Supabase | 두 세션 비교(애착·MBTI 궁합) → 바이럴 K팩터. IQ엔 이미 compare 존재. |
| **브랜드 콜라보 광고** | 영업 | 푸망 모델 — 결과 페이지에 브랜드 쿠폰. |

## 4. 다음 우선순위 (운영자 액션 후)

1. **배포 후 즉시**: Google Search Console에 `sitemap.xml` 제출 → 76개 유형 페이지 색인 시작
2. **토스 승인 후**: 성격 테스트 유료 리포트 + 환불 보장 카피
3. **트래픽 확보 후**: 결과 페이지 AdSense (무료 사용자 수익화)
4. **Supabase 후**: 결과 저장 + 친구 궁합 매칭(바이럴) + 이메일 수집(리텐션)

## 5. 측정 지표 (배포 후 추적)

- **완주율** (테스트 시작 → 결과 도달) — 공유의 관문
- **공유율** (공유/완주) + **K팩터** (신규 유입/기존 사용자) — 1 넘으면 자가 성장
- **페이지/세션** — 목표 3+ (thicket.sh 5.3 벤치마크)
- **유형 페이지 색인율** (색인/제출) — 목표 60일 내 60%+
- **오가닉 유입** (유형 페이지 롱테일 키워드)
