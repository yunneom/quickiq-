# IQ Test Questions

이 파일은 **사람이 직접 작성하는 콘텐츠**입니다. Claude Code는 이 파일을 읽어 `scripts/seed.ts`에 자동으로 반영합니다.

## 구성 원칙

- **총 30문항** (한국어 30 + 영어 30, 총 60문항)
- 영역 균등: 언어 8 / 수리 8 / 공간 7 / 논리 7
- **대상 수준: 중학생 이상 성인.** 초등 수준 문항(단순 유의어·반대말, 2·4·6·8 수열 등) 금지.
- 난이도 분포(현재 base 30): 난이도3 11 / 난이도4 11 / 난이도5 8. **난이도 1·2(쉬움)는 사용하지 않음** — 바닥이 "생각해야 풀리는" 3부터 시작.
- 정답 위치는 A/B/C/D에 고르게 분산(현재 A8·B8·C8·D6)해 위치 추측을 방지.
- **출처**: 퍼블릭 도메인(고전 IQ 측정 문헌) + 자체 제작
- **금지**: Raven's Progressive Matrices 등 저작권 보호 검사의 원본 문항 복제
- **표현**: "추정 IQ / Estimated IQ"로 일관되게 표기

## 작성 포맷 (예시)

각 문항은 다음 포맷으로 작성:

```yaml
- order_index: 1
  category: verbal
  difficulty: 2
  locale: ko
  question_text: "다음 중 나머지와 종류가 다른 것은?"
  options:
    - { id: A, text: "사과" }
    - { id: B, text: "배" }
    - { id: C, text: "당근" }
    - { id: D, text: "포도" }
  correct_id: C
  explanation: "당근은 채소이고, 나머지는 모두 과일입니다."
```

## 영역별 문항 가이드

### 언어 추론 (Verbal) — 8문항
- 단어 관계 (유의어, 반의어, 범주)
- 문장 완성
- 단어 분류 ("나머지와 다른 것")
- 어휘 추론

### 수리 추론 (Numerical) — 8문항
- 수열 (등차, 등비, 피보나치 등)
- 간단한 응용 문제 (비율, 평균)
- 패턴 (숫자 매트릭스)
- 산술 추론

### 공간 추론 (Spatial) — 7문항
- 도형 회전/뒤집기
- 전개도/접기
- 패턴 완성 (이미지 필요)
- 좌우/거울상

### 논리 추론 (Logical) — 7문항
- 삼단논법
- 조건 추론 ("만약 ~라면")
- 순서/배열 문제
- 진위 판별

## 영어 버전

각 한국어 문항에 대응하는 영어 문항 필요. **단순 번역이 아니라 영어권 문화에 맞게 재작성** (예: 한국 속담 → 영어 격언).
