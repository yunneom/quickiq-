# Test Suite Roadmap — 신규 테스트 10개 후보

> **상태**: DRAFT — 성인용 IQ 매출 검증 후 선택 1–2개 착수
> **목적**: 같은 인프라(랜딩 → 무료 응시 → 결과 → PDF 결제 → 이메일)로 확장 가능한 호기심 자극형 테스트 후보
> **공통 조건**: 공신력 있는 공개 자료 기반, 법적 리스크 낮음, 인스타 광고 친화적, 5–10분 응시

---

## 평가 기준

각 후보를 5개 축으로 점수 (1–5):
- **CW (호기심 Curiosity)**: 광고 보면 클릭하고 싶은가
- **VR (바이럴 Viral)**: 결과를 친구에게 공유하고 싶은가
- **PW (결제 의사 Pay Willingness)**: 결과 페이지에서 PDF 결제 누를 가능성
- **LR (법적 리스크 Legal Risk)**: ↓ 낮을수록 좋음 (점수 거꾸로 = 5가 가장 안전)
- **DI (구현 난이도 Dev Intensity)**: ↓ 낮을수록 좋음 (점수 거꾸로 = 5가 가장 쉬움)

**합산 = CW + VR + PW + LR + DI** (최대 25점)

## 후보 10개

### 1. EQ 테스트 (감성지능) ⭐
- **개요**: Mayer-Salovey-Caruso 모델 단순화. 30문항으로 "감정 인식·이해·조절·활용" 4영역 점수
- **타겟**: 20–40대, 자기개발에 관심 있는 직장인
- **콘텐츠 출처**: Goleman의 EQ 프레임워크 (개념 공개), 자체 시나리오 문항
- **차별화**: "당신은 공감형 vs 분석형 리더" 식 시각화
- **광고 카피 예시**: "나의 EQ는 상위 몇 %일까? (7분 무료 테스트)"
- **점수**: CW 5 / VR 4 / PW 4 / LR 5 / DI 4 = **22**

### 2. Big Five 성격 검사 (5요인 모델) ⭐⭐
- **개요**: 외향성·친화성·성실성·신경성·개방성 5축 측정. 학계에서 가장 신뢰받는 성격 모델
- **타겟**: MBTI에 식상해진 사용자, 자기이해 욕구
- **콘텐츠 출처**: **IPIP (International Personality Item Pool) — 완전 퍼블릭 도메인**, 50문항 단축형 사용
- **차별화**: MBTI보다 과학적, 무료 응시 시점에 1축 결과 + 결제 시 5축 상세 + 직업·연애 매칭
- **광고 카피**: "MBTI보다 정확한 성격 5요인 — 7분 무료"
- **점수**: CW 4 / VR 5 / PW 5 / LR 5 / DI 5 = **24**

### 3. 직업 적성 / 진로 유형 (Holland RIASEC) ⭐⭐
- **개요**: Realistic·Investigative·Artistic·Social·Enterprising·Conventional 6축. 진로 적성 표준 모델
- **타겟**: 대학생, 이직 고민하는 20–30대 직장인, 학부모(자녀 진로용)
- **콘텐츠 출처**: **Holland code — 공개 모델**, O*NET 직업 매칭 데이터도 무료
- **차별화**: 결과 → 추천 직업 30개 리스트 + 한국 채용시장 평균 연봉 매핑 (PDF에 포함)
- **광고 카피**: "당신에게 딱 맞는 직업 TOP 30 — 무료 7분 테스트"
- **점수**: CW 5 / VR 3 / PW 5 / LR 5 / DI 4 = **22**

### 4. 회복탄력성 / 멘탈 강도 테스트 (Resilience)
- **개요**: Connor-Davidson Resilience Scale 단축형 응용. "역경 극복 능력" 점수화
- **타겟**: 번아웃 직장인, 자기관리 관심 20–40대
- **콘텐츠 출처**: CD-RISC 학술 공개 항목 (10문항 단축형), 자체 시나리오 보완
- **차별화**: 결과 + "회복탄력성 키우는 7일 챌린지" 가이드 (PDF)
- **광고 카피**: "당신의 멘탈 강도는? 회복탄력성 지수 무료 측정"
- **점수**: CW 4 / VR 3 / PW 4 / LR 5 / DI 5 = **21**

### 5. 16 성격 유형 테스트 (MBTI 대체) ⭐⭐⭐
- **개요**: Jungian 4차원 (E/I, S/N, T/F, J/P) — **"MBTI"라는 상표명 사용 금지**, "16가지 성격 유형" 표현 사용
- **타겟**: 10–30대 (한국에서 폭발적 관심)
- **콘텐츠 출처**: Keirsey Temperament + Jung 원형 (개념 공개), 자체 문항
- **차별화**: "당신의 연애 궁합 유형 ↔ 매칭" 같은 viral hook + PDF에서 16x16 궁합표
- **광고 카피**: "MBTI는 알지만... 진짜 나는 어떤 유형? (정밀 분석)"
- **⚠️ 주의**: "MBTI", "Myers-Briggs" 상표 절대 사용 금지. "16 personality types" / "성격 유형 분석"으로
- **점수**: CW 5 / VR 5 / PW 4 / LR 3 / DI 5 = **22**

### 6. 연애 스타일 / 애착 유형 테스트 ⭐⭐
- **개요**: Hazan & Shaver의 성인 애착 이론 (안정·불안·회피·혼란 4유형). "왜 나는 늘 그런 사람을 만나는가"
- **타겟**: 20–30대 미혼·재혼, 연애 고민 사용자
- **콘텐츠 출처**: ECR-R (Experiences in Close Relationships - Revised) 학술 공개
- **차별화**: 무료 결과 + "당신과 잘 맞는 애착 유형" 매칭 + 친구·연인 비교 기능
- **광고 카피**: "왜 늘 비슷한 연애를 반복할까? 애착 유형 무료 분석"
- **VR 끝판왕**: 친구·연인 코드 공유로 두 사람 매칭 → 바이럴 폭발 가능
- **점수**: CW 5 / VR 5 / PW 4 / LR 5 / DI 4 = **23**

### 7. 창의력 테스트 (Divergent Thinking)
- **개요**: Guilford의 Alternative Uses Test 응용. "벽돌의 사용처를 최대한 많이 적으세요" 등
- **타겟**: 디자이너·기획자·창업 관심 사용자
- **콘텐츠 출처**: Guilford·Torrance 공개 프레임워크
- **차별화**: AI 평가 (OpenAI API) 필요 → 운영비 증가 단점
- **광고 카피**: "당신의 창의력 지수는? 디자이너·기획자 평균 비교"
- **⚠️ 운영비**: AI 채점 비용 발생 (응시당 ~₩50–100)
- **점수**: CW 4 / VR 3 / PW 4 / LR 5 / DI 2 = **18**

### 8. 좌뇌-우뇌 우세도 테스트
- **개요**: 좌/우뇌 우세를 시각·언어·논리 vs 직관·예술 영역 점수로
- **타겟**: 가벼운 호기심 사용자, 10–30대
- **콘텐츠 출처**: 자체 제작 (학계에선 사실 신화로 분류, 엔터테인먼트 분명히)
- **⚠️ 주의**: "**Just for fun · 과학적 진단 아님**" 강한 disclaimer 필수
- **차별화**: 진단력은 낮지만 응시 부담 낮고 SNS 공유율 높음
- **광고 카피**: "당신은 좌뇌형? 우뇌형? 1분 테스트"
- **점수**: CW 4 / VR 5 / PW 2 / LR 3 / DI 5 = **19**

### 9. 리더십 유형 / 보스 스타일 테스트
- **개요**: Goleman의 6가지 리더십 스타일 (지시·민주·코칭·관계·페이스·비전형) 매핑
- **타겟**: 팀장·관리자·예비 창업자
- **콘텐츠 출처**: Goleman 공개 프레임워크
- **차별화**: 결과 + "당신 팀원이 본 당신" 360도 평가 옵션 (친구 코드 공유)
- **광고 카피**: "당신은 어떤 보스인가? 6가지 리더십 유형 진단"
- **점수**: CW 3 / VR 3 / PW 4 / LR 5 / DI 4 = **19**

### 10. 색채 성격 테스트 (Color Personality)
- **개요**: 색상 선호도 → 성격 유형 매핑 (Lüscher 색채 검사 변형, 단순화)
- **타겟**: 10–20대 여성 (인스타 친화)
- **콘텐츠 출처**: 자체 제작 (Lüscher 원본은 라이선스, 컨셉만 차용)
- **차별화**: **시각적으로 인스타 친화** — 결과 이미지가 곧 OG 콘텐츠
- **광고 카피**: "당신의 성격을 색깔로 표현한다면? (재미로 보는 테스트)"
- **⚠️ 주의**: "재미용 · 과학적 검사 아님" 명시
- **점수**: CW 4 / VR 5 / PW 3 / LR 4 / DI 4 = **20**

## 운영 전략 — 단일 브랜드 vs 멀티 브랜드

### 옵션 A: 단일 브랜드 + 멀티 테스트 ⭐ 추천
같은 도메인 (`quickiq.kr` 또는 새 도메인)에 여러 테스트 메뉴.

```
/                       # 테스트 선택 허브
/iq                     # IQ 테스트
/eq                     # EQ 테스트
/big-five               # Big Five
/attachment             # 애착 유형
...
```

**장점**:
- 운영 인프라 1개 (LS store, Supabase, Vercel)
- 한 사용자가 여러 테스트 응시 → LTV ↑
- "QuickTest" 같은 우산 브랜드로 확장

**단점**: 브랜드가 "IQ 테스트"에서 "심리 테스트 종합"으로 희석

### 옵션 B: 각 테스트별 독립 도메인
**장점**: SEO/광고 채널별 최적화 가능
**단점**: 운영 부담 N배

→ **A부터 시작, traction 보고 인기 테스트만 분리하는 게 정석**

## 추천 실행 순서 (성인 IQ 매출 발생 후)

### Phase 1: cross-sell 검증 (2주)
- **Big Five 추가** (가장 안전)
- 기존 IQ 응시자에게 "다음 테스트도 응시해보세요" 이메일
- 응시율·결제율 측정 → cross-sell 효과 검증

### Phase 2: 바이럴 확장 (4주)
- **애착 유형** 추가 (친구 매칭 hook으로 신규 유입)
- 인스타 광고: "친구랑 같이 풀어보는 연애 궁합 테스트"

### Phase 3: 시장 확장 (6주)
- **16 성격 유형** + **RIASEC** 추가
- "QuickTest" 우산 브랜드 출범 검토

### Phase 4 (선택): 엔터테인먼트
- **좌뇌-우뇌** / **색채 성격** — 광고 CTR 부스터용 진입 테스트
- 무료 응시 → 상위 테스트로 funnel

## 기술적 공통 인프라 (재사용 가능 부분)

| 자산 | 재사용 가능성 |
|---|---|
| 응시 UI 컴포넌트 (`components/test/*`) | ✅ 100% 재사용 |
| 채점 엔진 (`lib/scoring/`) | ⚠️ 테스트별 채점 로직 추가 필요 |
| Result 페이지 레이아웃 | ✅ 90% 재사용 (강점/약점 차트 등) |
| PDF 렌더링 (`lib/pdf/`) | ⚠️ 테스트별 템플릿 추가 |
| Lemon Squeezy 결제 flow | ✅ 100% 재사용 |
| Resend 이메일 발송 | ✅ 100% 재사용 |
| Supabase schema | ⚠️ `test_type` 컬럼 추가 |
| Meta Pixel funnel | ✅ 100% 재사용 (test_type 파라미터 추가) |

**핵심 인사이트**: IQ 테스트 인프라의 70%는 재사용 가능. 새 테스트 1개 추가 = 2–3주 (문항 작성 + 채점 로직 + PDF 템플릿 + 광고 크리에이티브).

## 의사결정 트리거

```
성인 IQ 매출 발생?
├── No  → 이 문서 봉인, IQ에 집중
└── Yes → 30일 누적 매출 100만원+?
         ├── No  → IQ 광고·가격 최적화 (다른 테스트 ㄴㄴ)
         └── Yes → Big Five 추가 (Phase 1) 시작
                  └── cross-sell 결제율 5%+?
                       ├── No  → Big Five 광고 단독 운영 검증
                       └── Yes → 애착 유형 추가 (Phase 2)
```

성인용 IQ가 "팔리는 상품" 임이 검증되기 전엔, 새 테스트 추가는 **모두 잡으려다 다 놓치는 함정**.
